# 네이버 쇼핑 검색 API 발급 가이드 (1분 소요)

이 가이드를 따라 하시면 매크로 차단 없이 평생 안전하게 랭킹을 검색할 수 있는 키(Key)를 2개 받을 수 있습니다.

**1. 네이버 개발자 센터 접속 및 로그인**
아래 링크를 클릭해서 평소 쓰시는 네이버 아이디로 로그인해 주세요.
👉 https://developers.naver.com/main/

**2. Application 등록하기**
상단 메뉴에서 `[Application]` 탭에 마우스를 올리고 `[애플리케이션 등록]`을 클릭하세요.

**3. 정보 입력하기**
- **애플리케이션 이름**: `오즈키즈 랭킹 검색기` (마음대로 적으셔도 됩니다)
- **사용 API**: 드롭다운을 눌러서 `[검색]`을 선택해 주세요.
- **비로그인 오픈 API 서비스 환경**: `[WEB 설정]`을 선택해 주세요.
  - 나타나는 **웹 서비스 URL** 칸에는 `http://localhost` 하나만 적으시면 됩니다.
- 제일 아래 초록색 `[등록하기]` 버튼 클릭!

**4. ⭕ 완료! 결과 복사하기 ⭕**
등록이 완료되면 화면 상단에 두 줄짜리 영어+숫자로 된 암호가 뜹니다!
이 두 개를 복사해서 채팅창에 그대로 붙여넣어 주세요!

1. `Client ID` : (예: AbCdEfG_1234567)
2. `Client Secret` : (예: xYzA123)

이 두 개만 주시면 제가 코드를 갈아끼고 바로 완성해 드리겠습니다!
